User Guide:

Welcome to Stock Genius! This application helps beginner and retail investors visualize stock performance with ease. Review the User Guide below to make the most of the application.

Navigation Bar: Same on each page, allows you to redirect to the Home Page, Help Page, and ABout Page.

Home Page: The Home Page Consists of 4 Main Features

  • Text Search: To search for a stock through text, click into the ticker box and search the stock symbol for example AAPL, MSFT, LCID etc. DO NOT SEARCH A STOCK BY COMPANY NAME ONLY TICKER. Once you input the stock ticker in the search box, click the lookup button and it will redirect you to the stocks page with the chart and data about the stock.

  • Speech Search: To search for a stock through speech, click the microphone button, and allow the mic to be turned on. Once the mic is turned on, the button will change to a mic without a cross through it indicating the mic is on. search the stock symbol for example AAPL, MSFT, LCID etc. DO NOT SEARCH A STOCK BY COMPANY NAME ONLY TICKER. Once you speak the stock ticker it will automatically redirect you to the stocks page with the chart and data about the stock.

  • Watchlist: The watchlist is personal to each user. Whenever a stock is searched, the user will have the option to click a button to add the stock to the watchlist. They can also remove the stock from the watchlist by clicking the same button. Note: If you view the application on a different browser, the watchlist will be different. For example, if you initialy added the stocks to the watchlist using a Safari browser, then view the application on a Chrome browser, then the watchlist WILL NOT populate with the stocks that were favorited on the Safari browser.

  • Recently Searched List: The Recently Searched List consists of the last 15 stocks searched by anyone who has searched a stock on the application. Whenever a stock is searched, it automatically gets added to the recently searched list.

Stocks Page: The Stocks Page Consists of 4 Main Features

  • Back Button: The back button redirects you back to the Home Page. You can also redirect back to the Home Page Using the Home button on the nav bar.

  • Favorites Button: The favorites button allows the user to add the stock to their watchlist. When the button is clicked, image changes to a yellow star, and adds the stock to the watchlist. If the star is transparent, then the stock is not in the watchlist. If you click on a the star when it is yellow, it will remove the stock from the watchlist. Additonally, if a stock is favorited, and you navigate to a different page, when you return to the favorited stock, it will still display a yellow star.

  • Stock Data: Once the stock page is loaded, stock data such as the stock ticker, current price, price change, percent change, low of day, high of day, and current price will all be displayed. If the price change is negative, all of this data will be red, if the price change is positive, all of this data will be green, if the price change is 0.00, then the data will be grey.

  • Stock Chart: Once the stock page us loaded, the a line chart will display. If you hover over the points on the line chart, it will display a pop up with the date and closing price on that day. If the price change is negative, the line chart data will be red, if the price change is positive, the line chart data will be green, if the price change is 0.00, then the line chart data will be grey.

Stock Splits: Reverse Split VS Forward Splits

  • Forward Splits: "A stock split is when a company increases the number of its outstanding shares to boost the stock's liquidity. Although the number of shares outstanding increases, there is no change to the company's total market capitalization as the price of each share is split as well" (Gratton, 2024).
  • Reverse Splits: "Reverse stock splits are the opposite, in which a company lowers the number of shares outstanding to raise its stock price" (Gratton, 2024).

Forward and Reverse Split Examples

  • Forward Split: 2-1 (2 for 1)
    • The number of shares outstanding double, if there were 10 million, there is now 20 million outstanding.
    • The price of the stock splits in half, if it was $10 it is now $5.
  • Reverse Split: 1-2 (1 for 2)
    • The number of shares outstanding split in half, if there were 10 million, there is now 5 million outstanding.
    • The price of the stock doubles, if it was $10 it is now $20.
